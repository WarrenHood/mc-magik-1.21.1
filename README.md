# Magik

This is a Minecraft NeoForge 1.21.1 modpack focused on magic and exploration.

## What to expect

- Adventure (The Aether, Twilight Forest, Deeper Darker, improved worldgen and more)
- Cobblemon (Pokemon in minecraft)
- Enhanced mob AI to make mobs smarter and more challenging
- Farmers Delight and several addons to integrate it with the rest of the pack (cooking and farming mod)
- Many magic/fantasy systems to get into
  - Forbidden and Arcanus (Rituals, Weapon/Tool upgrades, ore doubling)
  - Hexerei (Witchcraft, cauldron brewing, flying broomsticks for travel)
  - Iron's spellbooks (integrated with Cataclysm and Ice and Fire)
  - Malum (Spirit magic - harvest and use spirits from mobs)
  - Nature's Aura (Absorb aura from the environment for different purposes)
  - Occultism (Demon summoning, storage and automation)
  - Spectrum (Color magic and Schizophrenia)
  - Theurgy (Alchemy, ore quadrupling and some automation)
  - Vampirism (Become a vampire or a vampire hunter):
    - Bloodlines - skill trees for vampires and hunters
  - Ice and Fire: Community Edition (Dragons and other mythical creatures)
- Proximity chat

## Performance improvements

- A bunch of performance mods
- The `Async` mod allows entity processing to happen across multiple cores and threads. This should allow for many more entities to be loaded at a time while keeping the TPS stable (and hopefully without too many crashes/instability since this is an alpha mod)
